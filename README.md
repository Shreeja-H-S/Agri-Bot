# AgriBot-AI-powered-farming-assistant

AgriBot is an AI-powered chatbot designed to assist farmers by providing information on crop cultivation, fertilizers, pest management, weather guidance, and general agricultural practices.

## Features
- Crop-related guidance
- Fertilizer recommendations
- Pest and disease information
- Weather-based farming tips
- User-friendly chatbot interface

## Tech Stack
- Python
- Flask
- HTML, CSS, JavaScript
- Machine Learning / NLP

## How to Run
1. Clone the repository.
2. Install the required dependencies.
3. Run the application.
4. Open the local host URL in your browser.

## Future Enhancements
- Voice-based interaction
- Multi-language support
- Real-time weather integration
- Crop price prediction
